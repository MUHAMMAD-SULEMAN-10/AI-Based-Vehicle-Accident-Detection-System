## 📌 Project Overview
This project is a real-time AI-based vehicle accident detection system that uses deep learning and computer vision to identify road accidents from video streams and trigger instant alerts to improve emergency response time.

---

## ⚙️ Key Features
- Real-time accident detection using CNN model
- Live video processing with OpenCV
- Instant alert system for emergency response
- Flask-based web streaming dashboard
- MQTT-based communication system
- Android integration for notifications

---

## 🧠 Technologies Used
- Python
- TensorFlow / Keras
- OpenCV
- Flask
- MQTT (paho-mqtt)
- NumPy
- HTML/CSS (Frontend)
- Deep Learning (CNN)

---

## 📊 Model Performance
- CNN Model Accuracy: ~99%
- Real-time video inference optimized for live detection

## 🚀 Impact
This system aims to reduce emergency response time by detecting accidents instantly and enabling faster alerts to authorities and emergency services.
